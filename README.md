# GCAP Labs — Landing Page

Premium, cinematic marketing site for **GCAP Labs** and its AI agent products.

## Products

- **Headmaster** — Persistent AI agent for organizations. Powered by Hermes (open-source, MIT, by Nous Research). Features: persistent memory, auto-generated skills, 17 messaging platforms, 300+ AI models, natural language scheduling, subagent delegation, 5 sandbox backends, browser automation, image gen, TTS, web search, plugin system, fast mode, /steer mid-run nudges, backup/restore, local web dashboard.
- **Headmaster HQ** — Orchestration addon for teams of agents. Powered by Paperclip (open-source, MIT). Features: org chart with agent roles/titles/reporting lines, goal alignment, heartbeat execution, atomic task checkout, per-agent budget + cost control, governance with rollback, human approval gates, full audit trail, multi-org isolation, runtime skill injection, company portability.
- **TayX** — GCAP's own fine-tuned model layer. Coming soon.

## Design
- Dark, cinematic, editorial aesthetic
- Persistent animations and scroll reveals
- Focus on product proof and workflow demonstrations
- Trust signals through security, governance, and audit trails

## Tech
- Next.js 16.2.6 (App Router)
- TypeScript, React 19
- Tailwind CSS 4, Framer Motion 12
- Vercel Blob for image storage
- Lenis smooth scroll, custom cursor tracking
